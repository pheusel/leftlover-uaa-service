== How to use this service

==== Sign up

  curl -i -H "Content-Type: application/json" -X POST -d '{
      "username": "admin",
      "password": "password"
  }' http://localhost:8080/UAAService/signup

==== Log in

  curl -i -H "Content-Type: application/json" -X POST -d '{
      "username": "admin",
      "password": "password"
  }' http://localhost:8080/login
  
You will receive a JWT as output.
